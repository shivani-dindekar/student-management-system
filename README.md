# Student Management System

## Project Overview

Student Management System is a web application developed using ASP.NET MVC, C#, SQL Server, jQuery, JavaScript, and Bootstrap. The application helps manage student records efficiently.

## Technologies Used

* ASP.NET MVC
* C#
* SQL Server
* Entity Framework
* JavaScript
* jQuery
* Bootstrap

## Features

* Add Student
* Update Student Details
* Delete Student
* Search Student
* View Student List

## Project Structure

* Controllers
* Models
* Views
* Scripts
* Content

## How to Run

1. Open the solution in Visual Studio.
2. Restore NuGet packages.
3. Configure SQL Server connection string.
4. Run the application.
